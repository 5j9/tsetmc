 نام سرويس

BestLimitsAllIns
 توضيحات  تقاضاهاي برتر خريد و فروش همه نمادها را ارائه مي کند. 
 پارامترهاي ورودي
 UserName  نام کاربري مشترک
 Password  کلمه عبور مشترک
 Flow

 کد بازار

  • 0 : عمومي - مشترک بين بورس و فرابورس
  • 1 : بورس
  • 2 : فرابورس
  • 3 : آتی
  • 4 : پایه فرابورس
  • 5 : پایه فرابورس (منتشر نمی شود)

 اطلاعات خروجي  
 number  رديف - براي هر نماد تا 5 مظنه برتر نمايش داده مي شود که اين عدد معرف شماره رديف مظنه است.
 InsCode  کد داخلي نماد
 QTitMeDem  حجم تقاضا
 ZOrdMeDem  تعداد تقاضا
 PMeDem  قيمت تقاضا
 PMeOf   قيمت عرضه
 ZOrdMeOf  تعداد عرضه
 QTitMeOf   حجم عرضه
